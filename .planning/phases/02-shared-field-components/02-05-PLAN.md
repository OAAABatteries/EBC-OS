    **PortalTabBar.jsx**:
    - Props: `{ tabs, activeTab, onTabChange, maxPrimary = 4, t }`
    - `tabs` is an array of `{ id: string, label: string, icon: LucideComponent, badge?: boolean }`
    - Import: `MoreHorizontal` from `lucide-react` for the More tab icon
    - State: `const [sheetOpen, setSheetOpen] = useState(false)`
    - Logic:
      ```jsx
      const primaryTabs = tabs.slice(0, maxPrimary);
      const overflowTabs = tabs.slice(maxPrimary);
      const hasOverflow = overflowTabs.length > 0;
      const isActiveInOverflow = overflowTabs.some(tab => tab.id === activeTab);
      ```
    - Renders:
      ```jsx
      <>
        <nav className="field-tab-bar" role="tablist">
          {primaryTabs.map(tab => (
            <button
              key={tab.id}
              className={`field-tab-item focus-visible ${activeTab === tab.id ? 'active' : ''}`}
              role="tab"
              aria-selected={activeTab === tab.id}
              onClick={() => { onTabChange(tab.id); setSheetOpen(false); }}
            >
              <tab.icon size={20} aria-hidden="true" />
              <span>{t ? t(tab.label) : tab.label}</span>
              {tab.badge && <div className="field-tab-badge" />}
            </button>
          ))}
          {hasOverflow && (
            <button
              className={`field-tab-item focus-visible ${sheetOpen || isActiveInOverflow ? 'active' : ''}`}
              role="tab"
              aria-expanded={sheetOpen}
              onClick={() => setSheetOpen(prev => !prev)}
            >
              <MoreHorizontal size={20} aria-hidden="true" />
              <span>{t ? t("More") : "More"}</span>
            </button>
          )}
        </nav>

        {/* Bottom sheet overlay + panel */}
        {hasOverflow && sheetOpen && (
          <div className="field-tab-sheet-overlay" onClick={() => setSheetOpen(false)} aria-hidden="true" />
        )}
        {hasOverflow && (
          <div className={`field-tab-sheet ${sheetOpen ? 'open' : ''}`}>
            {overflowTabs.map(tab => (
              <button
                key={tab.id}
                className={`field-tab-sheet-item focus-visible ${activeTab === tab.id ? 'active' : ''}`}
                onClick={() => { onTabChange(tab.id); setSheetOpen(false); }}
              >
                <tab.icon size={20} aria-hidden="true" />
                <span>{t ? t(tab.label) : tab.label}</span>
                {tab.badge && <div className="field-tab-badge" />}
              </button>
            ))}
          </div>
        )}
      </>
      ```
    - Every tab item has icon + label (per D-07: never icon-only)
    - Every button includes `focus-visible` class for keyboard navigation
    - Bottom sheet slides up via CSS transition (`.field-tab-sheet.open` has `transform: translateY(0)`)
    - No hex literals. No inline colors.

    Export from barrel: add `export { PortalTabBar } from './PortalTabBar';` to index.js.
