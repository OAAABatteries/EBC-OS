
Replace it with these lines (per UI-SPEC §Interaction Contract):
```css
/* Typography utilities — consume tokens from tokens.css */
.text-sm{font-size:var(--text-sm)}.text-base{font-size:var(--text-base)}.text-lg{font-size:var(--text-lg)}.text-display{font-size:var(--text-display)}

.text-xs{font-size:var(--text-sm)}.text-md{font-size:var(--text-base)}.text-xl{font-size:var(--text-lg)}.text-2xl{font-size:var(--text-lg)}.text-3xl{font-size:var(--text-display)}
```

This maps:
- .text-sm -> var(--text-sm) = 11px (was 12px hard-coded — absorbs per UI-SPEC)
- .text-base -> var(--text-base) = 13px (NEW class)
- .text-lg -> var(--text-lg) = 18px (same value)
- .text-display -> var(--text-display) = 28px (NEW class)
- .text-xs -> var(--text-sm) = 11px (legacy alias, was 11px — no change)
- .text-md -> var(--text-base) = 13px (legacy alias, was 14px — absorbs per UI-SPEC)
- .text-xl -> var(--text-lg) = 18px (legacy alias, was 24px — absorbs per UI-SPEC)
- .text-2xl -> var(--text-lg) = 18px (NEW legacy alias)
- .text-3xl -> var(--text-display) = 28px (NEW legacy alias)

**Change 2: Add new utility classes block**

Add the following block AFTER the existing UTILITIES section (before the `/* == PHASE TRACKER == */` section, approximately after line 276):

